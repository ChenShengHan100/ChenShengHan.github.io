Reframing Long-tailed learning via Loss Landscape Geometry

Shenghan Chen*, [Other Authors]

Accepted by CVPR, 2026

[arXiv] [Code]

We investigate the head-tail seesaw dilemma from the loss landscape view. By transferring long-tailed learning into a continual learning task, this paper uses continuous learning and sharpness perception to improve model attention to tail classes without relying on external synthetic data.

Towards Refined Data Augmentation for Long-tailed Recognition

Shenghan Chen*, [Other Authors]

Manuscript in preparation

[Details]

We rethink data augmentation in Long-Tailed recognition, focusing on data distribution shift. Based on large models, we propose automated data augmentation to solve the bias problem caused by data imbalance and the mismatch between synthetic and raw data distributions.
